# api-bootstrap

A skeleton Clojure project for building a REST API. 

## Usage

Development:
```
$ lein repl
user => (go) # To start
user => (reset) # To reset system
```

To run the project from the command line:
```
lein run
```

To build a jar:
```
lein uberjar
```
